- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

## 🌐 Deployment

### GitHub Pages
1. Upload `portfolio.html` to your GitHub repository
2. Go to repository Settings > Pages
3. Select source as "Deploy from a branch"
4. Choose main branch and root folder
5. Your site will be available at `https://username.github.io/repository-name`

### Alternative Deployment
- **Netlify**: Drag and drop `portfolio.html` to Netlify
- **Vercel**: Upload file and deploy
- **Any Web Server**: Simply upload to any web hosting service

## 🎨 Customization

### Personal Information
Edit the following sections in `portfolio.html`:
- Name and title in hero section
- Contact information
- Social media links
- Professional experience timeline
- Research interests
- Publications list

### Color Scheme
Modify CSS variables in the `<style>` section:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
    /* Add your custom colors */
}
```

### Fonts
The portfolio uses Google Fonts:
- **Inter**: For body text
- **Merriweather**: For headings
